Кортежи и множества

Кортеж

упорядоченная коллекция элементов, которая может содержать элементы различных типов

кортежи могут использоваться для хранения данных, которые не должны изменяться

mixed_tuple = (1, 3.14, 'hello', True)

Отличия от списка

  • Создаются с помощью ()
  • Иммутабельные (неизменяемые)

a_tuple = (1, 2, 3)

a_tuple[2] = 4
# Returns: TypeError: 'tuple' object does not support item assignment

Методы кортежей

count(value)

возвращает количество вхождений значения в кортеже


my_tuple = (1, 2, 2, 3)
print(my_tuple.count(2))  # Вывод: 2
index(value)

возвращает индекс первого вхождения значения в кортеже


my_tuple = (1, 2, 2, 3)
print(my_tuple.index(2))  # Вывод: 2

Набор (сет)

неупорядоченная коллекция уникальных элементов


my_set = {1, 2, 3, 'a', 'b', 'c'}

Отличия от списка

  • Создаются с помощью {}
  • Элементы наборов не имеют фиксированного порядка
  • Не допускают дубликатов

	my_set = {1, 2, 3, 'a', 'b', 'c'}

Создание наборов

с использованием фигурных скобок


my_set = {1, 2, 3, 'a', 'b', 'c'}
my_set_single = {1,}

с использованием функции set()


another_set = set([1, 2, 3, 4])

Методы наборов

add(element)

добавляет элемент в набор


my_set.add(4)
remove(element)

удаляет элемент из набора; вызывает ошибку, если элемент отсутствует


my_set.remove(1)

Методы наборов

discard(element)

удаляет элемент из набора, но не вызывает ошибку, если элемент отсутствует


my_set.discard(5)
intersection(other_set)

возвращает новый набор, содержащий только элементы, общие для обоих наборов


intersection_set = set1.intersection(set2)  # Вывод: {3}
